href="/wiki/Datei:Milovan_%C4%90ilas_Sremska_Mitrovica_1933.jpg" class="internal" title="vergrößern und Informationen zum Bild anzeigen"></a></div>Đilas als Häftling  in <a href="/wiki/Sremska_Mitrovica" title="Sremska Mitrovica">Sremska Mitrovica</a> (1933)</div></div></div>\n<p>Đilas war Sohn eines begüterten Bauern und Polizeioffiziers. Er studierte ab 1929 <a href="/wiki/Philosophie" title="Philosophie">Philosophie</a> und <a href="/wiki/Rechtswissenschaft" title="Rechtswissenschaft">Rechtswissenschaften</a> an der <a href="/wiki/Universit%C3%A4t_Belgrad" title="Universität Belgrad">Universität</a> <a href="/wiki/Belgrad" title="Belgrad">Belgrad</a> und trat 1932 der damals <a href="/wiki/Legalit%C3%A4t#Illegalität" title="Legalität">illegalen</a> <a href="/wiki/Bund_der_Kommunisten_Jugoslawiens" title="Bund der Kommunisten Jugoslawiens">Kommunistischen Partei Jugoslawiens</a> (KPJ) bei. Von 1933 bis 1936 inhaftiert, wurde er 1938 Mitglied des <a href="/wiki/Zentralkomitee" title="Zentralkomitee">Zentralkomitees</a> (ZK) und 1940 des <a href="/wiki/Politb%C3%BCro" title="Politbüro">Politbüros</a> der KPJ.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>\n</p><p>1941 nahm Đilas an der Vorbereitung und der Organisation des bewaffneten Aufstandes gegen die <a href="/wiki/Italienischer_Faschismus" title="Italienischer Faschismus">italienischen Faschisten</a> in Montenegro teil. Er war während des <a href="/wiki/Zweiter_Weltkrieg" title="Zweiter Weltkrieg">Zweiten Weltkrieges</a> Mitglied ihres so genannten <i>Obersten Stabes</i> (<i>Vrhovni štab</i>) und hatte den Rang eines <a href="/wiki/Generalleutnant" title="Generalleutnant">Generalleutnants</a> der kommunistischen <a href="/wiki/Volksbefreiungsarmee_(Jugoslawien)" title="Volksbefreiungsarmee (Jugoslawien)">Volksbefreiungsarmee (NOV/POJ)</a>.<br />\nIm März 1944 reiste er als Teil einer jugoslawischen Militär- und Parteidelegation als Verbindungsmann nach <a href="/wiki/Moskau" title="Moskau">Moskau</a>.<sup id="cite_ref-convers_16-17._2-0" class="reference"><a href="#cite_note-convers_16-17.-2">&#91;2&#93;</a></sup> Dort traf er in Gesprächen unter anderem mit <a href="/wiki/Georgi_Dimitroff" title="Georgi Dimitroff">Georgi Dimitroff</a>, <a href="/wiki/Wjatscheslaw_Michailowitsch_Molotow" title="Wjatscheslaw Michailowitsch Molotow">Molotow</a> und auch mehrmals mit <a href="/wiki/Josef_Stalin" title="Josef Stalin">Stalin</a> zusammen.<sup id="cite_ref-convers_33-58_3-0" class="reference"><a href="#cite_note-convers_33-58-3">&#91;3&#93;</a></sup>\n</p><p>In der „Provisorischen Regierung“ vom 8. März 1945 (<a href="/wiki/Koalition_(Politik)" title="Koalition (Politik)">Koalitionsregierung</a> <a href="/wiki/Josip_Broz_Tito" title="Josip Broz Tito">Tito</a>-<a href="/wiki/Ivan_%C5%A0uba%C5%A1i%C4%87" title="Ivan Šubašić">Šubašić</a>) war Đilas <a href="/wiki/Minister" title="Minister">Minister</a> von <a href="/wiki/Montenegro" title="Montenegro">Montenegro</a>. Ab Februar 1946 war er <a href="/wiki/Minister_ohne_Gesch%C3%A4ftsbereich" title="Minister ohne Geschäftsbereich">Minister ohne Portefeuille</a> in verschiedenen Regierungen. Auf dem Parteikongress der KPJ vom 21. bis 28. Juli 1948 wurde Đilas in das <a href="/wiki/Zentralkomitee" title="Zentralkomitee">Zentralkomitee</a> und das <a href="/wiki/Politb%C3%BCro" title="Politbüro">Politbüro</a> der KPJ gewählt. Ab Januar 1953 war er Mitglied des <a href="/wiki/Pr%C3%A4sidium" title="Präsidium">Präsidiums</a> des <i>Bundes-Exekutivrats</i> (<i>Savezno Izvršno Veće, SIV</i>).\n</p><p>1954 kam es nach der Veröffentlichung einer 18-teiligen kritischen Artikelserie im Parteiorgan <i><a href="/wiki/Borba_(Zeitung)" title="Borba (Zeitung)">Borba</a></i> zum Bruch mit <a href="/wiki/Josip_Broz_Tito" title="Josip Broz Tito">Tito</a> und dem <a href="/wiki/Bund_der_Kommunisten_Jugoslawiens" title="Bund der Kommunisten Jugoslawiens">Bund der Kommunisten Jugoslawiens</a> (<i>Savez Komunista Jugoslavije</i>, <i>SKJ</i>), der Nachfolgeorganisation der KPJ ab 1952. Das Zentralkomitee des SKJ erhob gegen Đilas den Vorwurf des parteiwidrigen Verhaltens und entfernte ihn von allen Funktionen in Partei und Staat. Der Grund waren seine realsoziologischen Analysen, in denen er die kommunistischen <a href="/wiki/Kader" title="Kader">Kader</a> mit <a href="/wiki/Marxismus" title="Marxismus">marxistischen</a> Analysemethoden als <i>Neue <a href="/wiki/Soziale_Klasse" title="Soziale Klasse">Klasse</a></i> herausarbeitete und verurteilte. Wegen „Stellungnahmen gegen die jugoslawischen Interessen“ wurde er zu zwei weiteren Haftstrafen verurteilt und verbrachte die Jahre 1956 bis 1961 sowie 1962 bis 1966 im Gefängnis. from bs4 import BeautifulSoup

Lass dir nur den Text des Artikels ausgeben.

# Code

'''
Eine der wichtigsten Funktionen von BeautifulSoup ist wohl find_all.
Damit lässt sich nach bestimmten HTML-Tags suchen.
Die wichtigsten HTML-Tags sind:
<p> Paragraph </p>
<a> Hyperlink (Anchor) </a>
<ol> geordnete Liste </ol>
<ul> ungeordnete Liste </ul>
<li> Listeneintrag </li>

In diesem Fall wissen wir, dass der gesamte Text in dieser Webseite innerhalb von einem Parargraph-Tag steht.
'''
text = ''
for p in soup.find_all('p'):
    text += p.text
#text

soup.get_text()

Filtere alle Hyperlinks heraus.

# Code

Mithilfe der get Funktion können wir auf Attribute der jeweiligen Tags zugreifen.
So können wir z.B. die verlinkten Webseiten in einer Seite finden in dem wir uns das href-Attribut eines Anchor-Tags geben lassen.
'''
for link in soup.find_all('a'):
    print(link.get('href'))
